# Memory Assistant

一个 Chrome 扩展，自动捕获 AI 对话、构建分层记忆，并支持跨平台记忆导入。

## 支持平台

| 平台 | 域名 |
|---|---|
| ChatGPT | chatgpt.com / chat.openai.com |
| Google Gemini | gemini.google.com |
| DeepSeek | chat.deepseek.com |
| 豆包 | www.doubao.com |

---

## 安装

### 1. 获取 DeepSeek API Key

前往 [platform.deepseek.com](https://platform.deepseek.com) 注册并创建 API Key，用于后台记忆提炼。

### 2. 在 Chrome 中加载扩展

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的**开发者模式**
3. 点击**加载已解压的扩展程序**
4. 选择 `memory_assistant` 文件夹
5. 扩展图标将出现在工具栏中

### 3. 首次配置

1. 点击扩展图标，打开弹窗
2. 点击 **设置** → 填入 DeepSeek API Key → 点击 **保存**
3. 点击 **选择目录** → 选择一个本地文件夹用于保存记忆文件

## 使用说明

### 自动捕获（推荐）

扩展可在后台静默捕获每次对话并自动处理。

1. 点击扩展图标
2. 开启 **保持更新** — 扩展开始在所有支持的平台上捕获对话
3. 开启 **实时更新** — 每轮 AI 回复后，自动调用 DeepSeek 更新记忆（用户画像、偏好、项目进展、情节记录）

开启后正常聊天即可，无需任何手动操作，记忆自动积累。

> **提示：** 后台积累的记忆暂存于浏览器本地存储中。随时点击 **同步** 按钮，将其写出为 JSON 文件保存到本地文件夹。

---

### 同步到文件

点击显示目录名称的 **同步** 按钮，将浏览器存储中的全部记忆写出到本地文件夹。同步时会自动清理重复的对话记录。

写出的文件结构：

```
<选择的目录>/
├── profile.json          # 用户画像
├── preferences.json      # 回复偏好
├── workflows.json        # 工作流程模式
├── projects/
│   └── {项目名}.json     # 各项目笔记与决策记录
├── episodes/
│   └── {id}.json         # 每次对话的记忆快照
├── raw/
│   └── {平台}/
│       └── {会话id}.json # 原始捕获的对话记录
└── js_persistent_nodes.json  # 跨会话提炼的稳定规律
```

---

### 手动导出

针对当前对话进行一次高质量、结构化的记忆快照：

1. 在支持的平台上进行对话
2. 点击扩展图标 → **导出并保存记忆**
3. 扩展向 AI 注入结构化提取 prompt，等待 AI 回复后，调用 DeepSeek 更新持久记忆节点

适合在重要对话结束后使用，捕获更完整、精准的记忆。

---

### 构建持久记忆节点

持久节点是从多次对话中提炼出的跨会话稳定规律（如"偏好简洁回答"、"正在进行项目 X"等）。

**首次使用 / 积累了多次新对话后：**

点击 **从历史对话重建记忆节点**。扩展读取所有已捕获的 episodes，跳过已处理的部分，调用 DeepSeek 提炼稳定规律。状态栏会显示处理进度。

**合并相似节点：**

点击 **整理节点（合并相似）**，让 DeepSeek 找出语义重复的节点并合并（例如将同一课程下的多个知识点合并为一个学科节点）。

---

### 跨平台导入记忆

1. 在任意支持的平台上开始新对话
2. 点击扩展图标 → **按标签导入记忆**
3. 弹窗中按类型（偏好、画像、主题等）列出所有持久节点，置信度高或导出优先级高的节点默认勾选
4. 勾选需要导入的节点
5. 点击 **确认导入选中节点**

扩展将选中的节点及其对应的原始 episode 证据打包上传给 AI，并发送冷启动指令，使 AI 获得完整的历史上下文。

## 典型使用流程

```
日常聊天
  └─ 开启「保持更新」+「实时更新」→ 记忆自动积累

每次聊天后
  └─ 点击「同步」→ 更新本地文件夹中的记忆文件

定期维护
  └─ 点击「从历史对话重建记忆节点」→ 将新 episodes 提炼为持久节点
  └─ 点击「整理节点」→ 合并语义重复的节点

在新平台开始对话时
  └─ 点击「按标签导入记忆」→ 勾选节点 → 确认导入
```

## 常见问题

| 问题 | 解决方法 |
|---|---|
| 捕获没有触发 | 开启「保持更新」后刷新页面，扩展需要重新注入到页面中 |
| 按钮点击无响应 | 目标平台可能更新了页面结构，尝试重新加载扩展后再试 |
| 提示"API Key 未配置" | 点击「设置」重新填入 DeepSeek API Key |
| 提示"权限被拒绝" | 再次点击目录按钮重新授权，浏览器重启后可能撤销目录访问权限 |
| DeepSeek 接口报错 | 检查 API Key 是否有效，以及账户余额是否充足 |
| 文件中有重复对话记录 | 点击「同步」，同步时会自动去重并写出干净的数据 |
